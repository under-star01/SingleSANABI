**Single Sanabi**는 SANABI의 와이어 액션과 이동 시스템을 중심으로 구현한  
**2D 액션 프로젝트**입니다.

플레이어는 사슬팔을 활용해 이동하고,  
가속, 차지어택, 대시 어택 등 원작의 핵심 액션을 체험할 수 있도록 구성했습니다.

---

## 프로젝트 개요

- **프로젝트명** : Single Sanabi
- **장르** : 2D 액션 / 플랫포머
- **개발 인원** : 개인 프로젝트
- **개발 환경** : Unity, C#

---

## 주요 플레이 요소

- 사슬팔 기반 와이어 액션
- 가속, 차지어택, 대시 어택 구현
- 훈련장 기반 액션 연습 구조

---
Single SANABI 프로젝트의 실행용 빌드입니다.

## 실행 방법

- `SingleSANABI_Build_v1.0.zip` 다운로드
- 압축 해제
- `2DProject_SANABI.exe` 실행
